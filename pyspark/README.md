# PySpark Track

Platform: Databricks Community Edition (Spark Playground)
Notebooks go in `spark-playground/` and `concepts/`.

## Roadmap

### Phase 1 — Core Concepts
- [ ] SparkSession, SparkContext basics
- [ ] RDDs vs DataFrames — when to use which
- [ ] Reading data: CSV, JSON, Parquet
- [ ] DataFrame operations: select, filter, withColumn, drop

### Phase 2 — Transformations & Actions
- [ ] GroupBy, agg functions (count, sum, avg, max, min)
- [ ] Joins (inner, left, broadcast joins)
- [ ] Window functions in PySpark
- [ ] UDFs — when to use and pitfalls

### Phase 3 — Performance & Real-World
- [ ] Partitioning and repartition vs coalesce
- [ ] Caching and persistence strategies
- [ ] Handling skewed data
- [ ] Writing optimized output (partitioned Parquet)
