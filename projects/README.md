# Projects

End-to-end mini projects that tie SQL, PySpark, and Python together.
Each project gets its own folder with a `README.md` describing the goal and what was built.

## How to Add a Project

1. Create a folder: `projects/<project-name>/`
2. Add a `README.md` with: goal, dataset used, steps, what you learned
3. Add your code/notebooks
4. Update the table above with status
